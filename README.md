# News Application

A modern web application for reading news articles from various categories. Users can browse, navigate, and read the latest news with a clean and responsive interface.

---

## Features

- Browse latest news articles  
- Navigate through multiple categories  
- Responsive design for mobile, tablet, and desktop  
- Smooth navigation with React Router  
- Clean UI built with TailwindCSS  
- Lightweight and fast using Vite  

---

## Tech Stack

- **Frontend:** React + Vite  
- **Styling:** TailwindCSS  
- **Routing:** React Router DOM  
- **Build Tool:** Vite  
- **Linting:** ESLint
